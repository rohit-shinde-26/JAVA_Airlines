## ▶️ Getting Started

Clone the repository:

```bash
git clone https://github.com/rohit-shinde-26/JAVA_Airlines.git
```

Go to the project directory:

```bash
cd JAVA_Airlines
```

Compile a Java file:

```bash
javac FileName.java
```

Run the program:

```bash
java FileName
```
